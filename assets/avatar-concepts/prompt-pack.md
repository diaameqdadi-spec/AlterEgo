# Robot Avatar Prompt Pack

Use these prompts to generate consistent robot/avatar still assets for Alter Ego.

## Global Style Rules

- premium robotic character still
- dark or transparent background
- clean centered composition
- 3/4 view or front-facing
- strong silhouette
- high detail in face plate, visor, panel lines, and emissive accents
- no text
- no watermark
- no human skin
- no cartoon toy proportions unless explicitly desired
- polished product-shot lighting

## Shared Base Prompt

```text
Premium robotic avatar concept for a futuristic AI identity platform, highly detailed faceplate, sleek visor, segmented armor, elegant mechanical paneling, refined silhouette, centered composition, studio product-shot lighting, dark minimal background, sharp edges, premium materials, high contrast, clean presentation, no text, no watermark
```

## Negative Prompt Guidance

```text
blurry, low detail, text, watermark, extra limbs, toy-like plastic, childish proportions, human skin, organic face, cluttered background, overexposed lighting, low-quality render
```

## Selection Criteria

Choose finalists based on:

1. strong silhouette at small size
2. readable visor/face area
3. premium mechanical detail
4. consistent framing across variants
5. color identity that remains distinct from other avatars
